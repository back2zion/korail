# KORAIL — 데이터 똑똑 관리 계획

**프로젝트 설명**
- **요약:** 한국철도공사(KORAIL)를 위한 데이터 관리 및 AI 준비를 목표로 한 '데이터 똑똑 관리 계획' 웹 자료입니다. 이 저장소의 `index.html`은 사업 개요, 현황 분석, 목표 모델, 이행 계획 및 발전 방안을 시각적으로 정리한 문서입니다.
- **기간:** 2025-07-14 ~ 2025-11-13 (4개월)
- **주체:** 한국철도공사 (KORAIL)
- **협력사:** 데이터스트림즈㈜

**핵심 목표**
- **데이터 거버넌스 확립:** 데이터 관리 조직(데이터 지키미 팀) 구성 및 역할 정의
- **데이터 표준화·품질 개선:** 데이터 정리 규칙과 품질 관리 수립
- **AI 준비:** 데이터 정제와 AI 운영 체계(AIMS) 준비
- **단계적 로드맵:** 2026~2028년 이행 계획 및 예산 제시

**문서(구성)**
- `index.html`에 다음과 같이 챕터별 내용이 정리되어 있습니다:
	- Chapter Ⅰ. 시작하는 이야기 (사업 개요, 추진 배경, 일정)
	- Chapter Ⅱ. 현황 분석 (문제점 및 해결 과제)
	- Chapter Ⅲ. 목표모델 설계 (목표 모델 및 AI 준비)
	- Chapter Ⅳ. 이행 계획 (이행 과제, 로드맵 및 예산)
	- Chapter Ⅴ. 발전 방안 (AI 자료 관리, 업무 용어사전 등)

**사용 방법**
- 로컬에서 문서를 확인하려면 브라우저로 `index.html`을 열거나 간단한 웹서버를 실행하세요:

```bash
# 현재 디렉터리에서 간단히 확인
# 브라우저로 파일 열기: 파일을 더블클릭하거나
# 로컬 서버 실행 (포트 8000):
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000/index.html 열기
```
